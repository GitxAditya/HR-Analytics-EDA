# HR Analytics Exploratory Data Analysis (EDA)

A complete end-to-end Exploratory Data Analysis (EDA) project built using Python to analyze HR data and generate actionable business insights. This project demonstrates data cleaning, exploratory analysis, visualization, statistical analysis, and business storytelling using real-world HR scenarios.

---

## Project Objective

The primary goal of this project is to analyze employee data to answer important business questions related to:

- Employee Salary Analysis
- Department Performance
- Promotion Trends
- Attrition Analysis
- Employee Performance
- Correlation Analysis
- Workforce Insights
- Executive Dashboard

The project focuses on transforming raw HR data into meaningful business insights that support data-driven decision-making.

---

## Dataset

The dataset contains employee information such as:

- Employee ID

## Key Business Insights

- Finance consistently has the highest average salary and performance.
- HR and Operations show higher employee attrition.
- Salary has a strong positive relationship with experience.
- Promotion decisions are largely aligned with employee performance.
- Top-performing employees are concentrated in a few departments.

---

## Business Risks

- High attrition in certain departments may increase recruitment costs.
- Uneven salary distribution can affect employee satisfaction.
- Heavy dependence on top-performing departments creates organizational risk.
- Talent concentration may impact succession planning.

---

## Recommendations

- Improve retention strategies for high-attrition departments.
- Review compensation policies across departments.
- Expand leadership development programs.
- Continue performance-based promotion policies.
- Build automated HR dashboards for continuous monitoring.
